## Project Components

### Data

Contains publicly available datasets obtained from the BLUEPRINT Epigenome Project and GEO, including:

- DNA methylation (bisulfite sequencing)
- RNA-seq from normal hematopoietic progenitors
- RNA-seq from AML samples
- RNA-seq from CLL samples

### Methods

Contains analysis scripts used throughout the project.

Major analytical modules include:

- DNA methylation analysis using RnBeads
- RNA-seq preprocessing and differential expression
- Integration of methylation and expression
- HOMER transcription factor motif enrichment
- Transcription factor clustering and visualization

### Results

Stores processed outputs generated by the analytical workflow, including:

- Differential methylation results
- Differential gene expression
- TF motif enrichment reports
- Heatmaps
- PCA plots
- TF clustering
- Integrated methylation-expression analyses

### Documentation

The `docs/` directory contains documentation describing:

- project background
- workflow
- analytical methods
- repository organization
- my individual contributions

## Notes

This repository preserves the original organization of the collaborative project while providing additional documentation to improve reproducibility and readability.
